# 3D_Maze

A first-person 3D maze game in the browser, built with Three.js (r128).
Each run generates a random maze via recursive backtracking; find the exit to win.

## Run

Static page — no build step. Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server
```

## Controls

- Click the game area to lock the mouse pointer
- `W` / `S` — move forward / back
- `A` / `D` — strafe left / right
- Mouse — look around
