# 3D Maze

This project is a simple first-person 3D maze game built with A-Frame. The maze is generated randomly each time the page loads and features a medieval theme with basic RPG elements like player health.

## Playing

Open `index.html` in a modern web browser or host the repository using GitHub Pages.

Use **WASD** keys to move and the mouse to look around.

## Building on the Game

Feel free to expand the maze size, add enemies or items, and customize the appearance. The goal is to provide a starting point for a medieval-themed 3D maze RPG.
